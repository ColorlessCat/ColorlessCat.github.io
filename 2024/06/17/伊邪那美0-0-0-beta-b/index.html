伊邪那美0.0.0_beta_b

巨大更新

先放链接：点我下载

更新内容：

1.增加了注册登录系统，可以收藏网络音声了。

2.修复了搜索功能，具体使用方式看后面。

3.UI优化，各种细节提升。

一些重要的事：

如果人机验证弹出的不是验证界面，先点击右上角的在线安装，然后重启软件即可。因为我用了腾讯的浏览器内核，它是这么设计的我没办法。

除了联网跟访问本地文件，剩下的权限都是腾讯要的，可以不给这些权限，不影响使用。

如果安装以后闪退，或者安装不上，需要先卸载老版本。

另外注册登录只做了最基本的功能，没有找回密码，所以尽量不要忘记密码。

然后是一些疑问：

Q.为什么显示网络错误？

A.需要科学上网。

Q.为什么按钮点击没反应？

A.因为我没写，很多按钮目前只是占位。

Q.怎么导入本地音声？

A.从Dlsite下载的音声，需要在根目录的名字带RJ号，然后本地扫描就可以自动导入。

Q.一播放就闪退。

A.已经重新编译了播放器，还是闪退可以去群里反馈。

Q.发现了BUG。

A.是的，这个版本BUG很多，可以去群里反馈。
