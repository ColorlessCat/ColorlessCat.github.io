DLPlayer-伊邪那美-0.0.1




介绍

DLPlayer是一个用于管理你从DLsite上下载的音声的软件，你也可以直接从网上在线播放音声。
下载链接：Dlplayer-0.0.1伊邪那美

特色

它可以根据Dlsite音声的文件格式，自动扫描手机上的音声并整理分类。并且支持搜索，以及网络在线播放。

初衷

我一直都在用某易云听音声，但是整理起来很不方便，日文也极度难找，于是萌生了自己写一个软件的想法。
同时，我用它来学习新的技术知识，顺便也想看看自己能不能长期维护一个项目。

计划

当前版本是-0.0.1，我对标的是商业级软件，但个人精力有限，因此我只把精力花在了主要界面与功能上，为了稳定性一些特性被我隐藏了。
尽管目前的版本很简陋，但已经足够使用了，后续就是在这个基础上慢慢添加细节。

支持

可以在爱发电上支持我，同时我其实还在找工作，如果有人能内推一下万分感谢！
爱发电

技术

简单描述一下本软件的技术栈，求内推：MVVM架构，UI使用纯Compose编写，Room数据库，Hilt依赖注入，Retrofit网络框架，coil图片加载框架等。项目熟练使用了kotlin协程+Flow，有一些自己实现的Composable：跑马灯Text，双进度进度条。
